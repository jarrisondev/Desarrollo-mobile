## Breve descripción del proyecto
CookBook es una aplicación móvil desarrollada con React Native para consultar y organizar recetas de cocina.
El proyecto permite visualizar listados de recetas y está orientado a practicar el desarrollo multiplataforma para Android e iOS.

## Instrucciones de instalación y ejecución
1. Clona el repositorio y entra a la carpeta del proyecto.
2. Instala dependencias de Node.js:

```bash
npm install
```

3. Si vas a ejecutar en iOS, instala dependencias nativas:

```bash
bundle install
npx pod-install
```

4. Inicia el servidor Metro:

```bash
npm start
```

5. Ejecuta la aplicación:

```bash
# Android
npm run android

# iOS
npm run ios
```

## Lista de tecnologías utilizadas
